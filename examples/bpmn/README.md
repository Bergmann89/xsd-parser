This example shows how to generate and use the code for the `BPMN 2.0` schema.

The original code of this example was provided by [Christopher22/bpmn-schema](https://github.com/Christopher22/bpmn-schema).

The XSD schemas for BPMN 2.0.2 were obtained from the [official OMG (Object Management Group) website](https://www.omg.org/spec/BPMN).
